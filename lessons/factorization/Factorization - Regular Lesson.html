Concept 1 - Factoring by Grouping

Grouping is useful when expressions have four or more terms. Typical forms are 2-2 grouping and 3-1 grouping.

Example 1 (from PDF style): solve for \(abc\)

Given \(a^2+2b^2+2c^2-2ab-2bc-6c+9=0\), find \(abc\).

Regroup into squares: \((a-b)^2+(b-c)^2+(c-3)^2=0\).
Each square is nonnegative, so each must be \(0\).
Hence \(a=b=c=3\), so \(abc=27\).
TYPE - 3-1 grouping into perfect squares
METHOD: Build a sum of squares. If it equals 0, every square term must be 0.

Example 2 (from PDF style): complete squares in two variables

Real numbers \(x,y\) satisfy \(x^2+y^2=10x-6y-34\). Find \(x+y\).

Move all terms: \(x^2-10x+y^2+6y+34=0\).
Complete squares: \((x-5)^2+(y+3)^2=0\).
So \(x=5\), \(y=-3\), and \(x+y=2\).

Concept 2 - Splitting & Adding Terms (Completing Squares)

Splitting terms: rewrite a term as a sum. Adding terms: add and subtract the same expression to create a factorizable pattern.

Example 3 (Math Exploration): factor \(a^2-4a+3\)

Factorize \(a^2-4a+3\).

Find two numbers with product \(3\) and sum \(-4\): \(-1,-3\).
Therefore \((a-1)(a-3)\).

Example 4 (PDF): complete square to factor \(x^4+4\)

Factorize \(x^4+4\) by adding and subtracting terms.

How to decide the added term
Goal: force a difference of squares: \((x^2+m)^2-(nx)^2\).
Expand: \((x^2+m)^2-(nx)^2=x^4+(2m-n^2)x^2+m^2\).
Match with \(x^4+4\): we need \(m^2=4\Rightarrow m=2\), and no \(x^2\)-term so \(2m-n^2=0\Rightarrow n^2=4\Rightarrow n=2\).
So we must create \((x^2+2)^2-(2x)^2\), which means adding and subtracting \(4x^2\).
Add and subtract \(4x^2\): \(x^4+4x^2+4-4x^2\).
Now \((x^2+2)^2-(2x)^2\).
Difference of squares gives \((x^2-2x+2)(x^2+2x+2)\).

Concept 3 - Simon's Favorite Factoring Trick (SFFT)

For forms like \(xy+ax+by\), add the product term to build two binomials:

\[xy+ax+by+ab=(x+b)(y+a)\]

Example 5 (Math Exploration): sum of possible \(x\)

Given positive integers \(x,y\) and \(xy-2x-3y=6\), find the sum of all possible values of \(x\).

Add 6: \((x-3)(y-2)=12\).
Positive divisor pairs for 12 give \(x-3\in\{1,2,3,4,6,12\}\).
So \(x\in\{4,5,6,7,9,15\}\), sum \(46\).

Example 6 (Math Exploration): all integer pairs

Given positive integers \(x,y\), solve \(6xy-10x-21y=49\).

Add 35: \((2x-7)(3y-5)=84\).
Check odd divisors for \(2x-7\): valid matches are \((2x-7,3y-5)=(3,28),(21,4)\).
Hence \((x,y)=(5,11),(14,3)\).

Concept 4 - Find Extreme Value by Completing Squares

Because perfect squares are nonnegative, writing an expression as a sum of squares plus a constant gives the minimum value immediately.

Example 7 (Math Exploration): two-variable minimum

Find the least value of \(2x^2-4xy+4y^2+8y+11\).

\(2x^2-4xy+4y^2+8y+11=2(x-y)^2+2y^2+8y+11\).
\(=2(x-y)^2+2(y+2)^2+3\).
Minimum is \(3\), reached at \(y=-2, x=-2\).

Example 8 (AMC 10A 2019 #19): product shift trick

Find the least value of \((x+1)(x+2)(x+3)(x+4)+2019\), where \(x\in\mathbb{R}\).

Pair factors: \((x^2+5x+4)(x^2+5x+6)+2019\).
Let \(t=x^2+5x\). Then expression is \((t+4)(t+6)+2019=(t+5)^2+2018\).
So least value is \(2018\).

H1Grouping / factor check

Which number is always a factor of \(3^{n+2}-2^{n+3}+3^n-2^{n+1}\)?

\(=10\cdot(3^n-2^{n+1})\), so the factor is 10.
H2Split middle term

Which factor must divide \(x^2-9x+8\)?

\(x^2-9x+8=(x-1)(x-8)\), so one factor is \(x-1\).
H3SFFT

How many reduced positive fractions \(\frac{x}{y}\) satisfy that \(\frac{x+1}{y+1}\) is 10% larger than \(\frac{x}{y}\)?

Equation gives \(10y(x+1)=11x(y+1)\Rightarrow y=\frac{11x}{10-x}\). Reduced positive solution is only \((x,y)=(5,11)\), so count 1.
H4Extreme value

Evaluate \(\dfrac{1^2-2^2+3^2-4^2+\cdots+99^2-100^2}{1+2+\cdots+100}\).

Numerator pairs to \(\sum_{k=1}^{50}((2k-1)^2-(2k)^2)=\sum_{k=1}^{50}(1-4k)=-5050\). Denominator is \(5050\). Value -1.
H5Completing square

Given \(x^2+x+1=0\), find \(x^3+2x^2+2x+2018\).

From \(x^2+x=-1\), expression \(=x^3+2(-1)+2018=x^3+2016\). Also \((x^2+x+1)(x-1)=x^3-1\Rightarrow x^3=1\). So 2017.
H6Squares

How many integer pairs \((x,y)\) satisfy \(x^{70}+y^2=2y\)?

\(x^{70}+(y-1)^2=1\). Cases: \(x^{70}=0\Rightarrow x=0,y=0\) or \(2\); \(x^{70}=1\Rightarrow x=\pm1,y=1\). Total 4 pairs.
H7SFFT

Positive integers satisfy \(xy-4x-5y=14\). Find the sum of all possible \(x\).

\((x-5)(y-4)=34\). Positive divisors of \(34\): \(1,2,17,34\), so \(x=6,7,22,39\). Sum 74.
H8Quadratic in \(x^2\)

For how many integers \(x\) is \(x^4-51x^2+50<0\)?

Let \(t=x^2\). Then \((t-1)(t-50)<0\Rightarrow 1<t<50\). Square values: \(4,9,16,25,36,49\), giving \(x=\pm2,\pm3,\pm4,\pm5,\pm6,\pm7\): 12 integers.
H9System + geometry

In an isosceles triangle, side lengths \(a,b\) satisfy \(9a^2-b^2=-13\), \(3a+b=13\). Find the third side.

From \(b=13-3a\): \(9a^2-(13-3a)^2=-13\Rightarrow a=2,b=7\). Isosceles side sets are \((7,7,2)\) or \((2,2,7)\); only first is valid. Third side is 7.
